# Qaida Mobile App

Flutter Android-приложение, собранное по HTML-референсам из папки design и гайду в docs/frontend/README.md.

Что уже есть:

- onboarding в стилистике дизайн-системы
- главная с discovery hero, карточками мест и bottom navigation
- многошаговый filter flow
- saved screen с pill-tabs и empty-state логикой
- notifications feed
- profile-заглушка под дальнейшую персонализацию
- экран деталей места

Запуск:

```bash
cd mobile_app
flutter pub get
flutter run
```

Проверка:

```bash
flutter analyze
flutter test
```
