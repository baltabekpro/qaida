# 🟡 Qaida — найди своё место в Алматы

> *"Скажи кто ты, с кем идёшь и сколько денег — Qaida скажет куда идти"*

Qaida — это сервис который решает одну простую боль: **не знаешь куда сходить в Алматы?** Ответь на 4 вопроса и получи 3 идеальных места за 30 секунд.

---

## 📱 Продукты

| Продукт | Технология | Описание |
|---|---|---|
| `qaida.kz` | Next.js | Веб-сайт |
| `@QaidaBot` | Telegram Mini App | Telegram бот |

---

## ⚙️ Как работает

```
Пользователь открывает Qaida
        ↓
Выбирает параметры (30 секунд):
  • С кем? → один / пара / друзья / семья
  • Бюджет? → до 3000₸ / до 7000₸ / без лимита
  • Халал?  → да / нет / неважно
  • 18+?    → да / нет
  • Сейчас открыто? → да / неважно
        ↓
Получает 3 лучших места с:
  фото · рейтинг · меню · адрес · время работы
        ↓
Нажимает "Идём" → открывается 2GIS / Google Maps
```

## 🗄 База данных

```
places         → заведения (название, адрес, координаты)
categories     → ресторан, кафе, бар, парк, кино...
filters        → халал, 18+, детская зона, парковка
working_hours  → пн-вс, время открытия/закрытия
menu_items     → блюда и цены
reviews        → отзывы пользователей
photos         → фотографии заведений
```

## 🚀 План запуска

```
Неделя 1-2  → База данных + бэкенд (FastAPI)
Неделя 3-4  → Веб-сайт (Next.js)
Неделя 5    → Telegram бот
Неделя 6    → Заполняем 50 заведений Алматы
Неделя 7    → Запуск, сбор отзывов
```
